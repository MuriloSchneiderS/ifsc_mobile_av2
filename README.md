# ifsc_mobile_av2
Segundo trabalho de mobile, plataforma online de publicação de livros.

Utilização do firebase para armazenar os arquivos.

Tela inicial com as últimas publicações e botão para criar  publicação, formulário para envio do arquivo/título/descrição e tela de  exibição do arquivo.
Se for possível utilizar giroscópio para estabilizar a exibição da página mesmo com movimentos e vibrações do dispositivo. Senão utilizar GPS para definir a localização do usuário quando fizer  cadastro, aí a foto de perfil vai ser uma bandeira do país.

Vamos tentar fazer com tipos de arquivo diversos, quero entender como é feito o armazenamento  de vídeos/imagens/áudios também, talvez ffmpeg seja necessário para  fazer a compressão e transcodificação dos arquivos em um formato mais  eficiente como webp/webm/epub.
